# Ollama
## Ollama란?
- LLM을 쉽게 실행할 수 있도록 하는 오픈소스 도구

## 설치
- For mac using Homebrew
```
brew install Ollama
```
## 모델 다운로드
- gemma4를 기준으로 예시

### Default(latest) 다운로드
```
ollama pull gemma4 or ollama pull gemma4:latest
```
### 특정 모델 다운로드
```
ollama pull gemma4:26b
```
